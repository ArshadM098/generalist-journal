This is a summary of the project I did for my Image Processing Class.
The method is an experimental approach at Image Denoising using Reinforcement Learning that creates an adaptive filter for the image.
The filter is trained using a reward function that is based on the difference between the original image and the denoised image.
The reward function is then used to train the filter using a Q-Learning algorithm. The filter is then applied to the image and the process is repeated until the reward function converges to a value that is close to zero. The filter is then applied to the image and the denoised image is displayed.
